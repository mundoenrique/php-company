Release notes for Kendo UI v2012.3.1315

Framework

What's new

  • Forced platform overrides appMode.
  • Zoom animation is broken with jQuery 1.7

DataSource

What's new

  • Add ability to check if DataSource has changes
  • Add support for canceling the requestStart event
  • Support for defining model fields as array

What's fixed

  • Can't specify custom parse, data and total when binding to xml
  • Fix Filtering on field which contains carriage return symbols
  • Sorting on negative numbers and zeros does not work as expected

Draggable

What's fixed

  • kendo.drag does not have destroy method

Globalization

What's new

  • Add support for escaped currency symbol in custom numeric format

What's fixed

  • kendo.parseFloat using Globalize.js throws exception when value is undefined
  • kendo.parseDate returns null if entered day is not valid

HierarchicalDataSource

What's fixed

  • Reading from a child datasource does not update the children field of the parent

Validator

What's new

  • Validate checkboxes on click

DataViz

Chart

What's new

  • Add select, selectStart and selectEnd events
  • Add series highlight configuration options

What's fixed

  • Chart tooltip is not displayed on tap
  • Series click and hover don't work.
  • Categories are not cleaned up after redraw/rebind
  • Drag event axis range is incorrect for a vertical/reversed value axis
  • value/x/y/Axis range can be incorrect when the default axis has a name
  • Stock chart can hang while zooming (w/mousewheel or touchpad) on OSX
  • Line/scatter series can't be clicked or hovered in IE 8
  • The chart prevents touch scrolling even when no drag events are attached
  • Pie chart with a segment without gradient doesn't show anything in IE 8.
  • Add missing category axis options - Justified, BaseUnitStep, AutoBaseUnitSteps, MaxGroups, RoundToBaseUnit, BaseUnit: Fit

Linear Gauge

What's fixed

  • Add data attributes for the radial and linear gauge.

Radial Gauge

What's fixed

  • Add data attributes for the radial and linear gauge.

StockChart

What's new

  • Add navigator dataSource, dateField and autoBind options
  • Add navigator visible and hint configuration settings
  • Add select, selectStart and selectEnd events

What's fixed

  • Stock chart can hang while zooming (w/mousewheel or touchpad) on OSX
  • Line/scatter series can't be clicked or hovered in IE 8
  • The chart prevents touch scrolling even when no drag events are attached
